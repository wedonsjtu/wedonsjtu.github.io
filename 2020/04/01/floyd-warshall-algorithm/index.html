Floyd-Warshall Algorithm - Shortest Path in Graph

Posted on 2020-04-01

Solution for all-pair shortest path problem in Graph.

Problem Description

Wikipedia: Floyd–Warshall algorithm

In a graph, find the minimum distance for every pair of nodes. The weights of edges can be positive or negative.

Algorithm

  1. Let V be the set of vertexes and E be the set of edges. V_num is the sizes of V. For every edge (u, v) in E, its weight is weight(u, v).
  2. define a matrix dist[V_num][V_num], whose values are initialized with "positive infinity" (You can use INT_MAX in C++).
  3. For every edge (u, v) in E, set dist[u][v] = weight(u, v).
  4. For every node v, set dist[v][v] = 0.

Time Complexity: \(\Theta ({|V|}^3)\); Space Complexity: \(\Theta ({|V|}^2)\).

Detection of negative cycles

We can use Floyd-Warshall algorithm to find the negative cycles.

Before setting dist[v][v] = 0, if dist[v][v] < 0, there is a negative self-cycle.

After finishing the algorithm, for any i, if dist[i][i] < 0, there is a negative cycle from i to i.
