Implement Stack using Queues; Implement Queue using Stacks

Posted on 2020-04-03

One Queue -> Stack; Two Stacks -> Queue

LeetCode Problem:

225. Implement Stack using Queues

232. Implement Queue using Stacks

Operations

Stack: push, pop, top (peek), empty (isEmpty)

Queue: push (add), pop (poll), front (peek), empty (isEmpty)

In C++ STL, pop returns nothing. In Java, pop returns the removed element.

See more: C++ Stack, C++ Queue, JDK 11 Stack, JDK 11 Queue.

To avoid confusion, we use some words: push_back, pop_front, push_top, pop_top.

Implement Stack using Queues

Use one queue to implement stack. Without the consideration of invalid operations.

Push

  1. Push_back the element x in the queue.
  2. If the front of queue isn't x, pop_front one element and push_back it again into the tail.
  3. Repeat Step 2 until x becomes the front of queue.

Pop

Pop_front one element from the queue. Return this element.

Empty

Check if the queue is empty.

Implement Queue using Stacks

Use two stacks (s1 and s2) to implement queue. Without the consideration of invalid operations.

Push

Push_top the element in s1.

Pop

  1. If s2 is not empty, pop_top one element from s2 and return this element.
  2. If s1 is not empty, pop_top one element from s1 and push_top it into s2.
  3. Repeat Step 2 until s1 is empty.
  4. Pop_top one element from s2. Return this element.

Empty

Check if both of two stacks are empty.
